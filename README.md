# Machine Learning Project

## Linear Regression & Logistic Regression (From Scratch)

This project implements Linear Regression and Logistic Regression using NumPy without using built-in models.

---

## 🔹 Question 1: Linear Regression

- Implemented using Gradient Descent
- Used Boston Housing dataset (scikit-learn)
- Calculated loss using Mean Squared Error
- Tested different learning rates
- Plotted loss vs iterations
- Compared results with scikit-learn model

---

## 🔹 Question 2: Logistic Regression

- Implemented Binary Classification using Sigmoid function
- Used Breast Cancer dataset (UCI)
- Loss function: Binary Cross Entropy
- Evaluated using Accuracy, Precision, Recall, F1-score
- Performed 5-fold Cross Validation
- Tested different classification thresholds

---

- Python
- NumPy
- Matplotlib
- scikit-learn (for comparison only)
